HOW TO USE FSH EMPIRE TRADING COMMAND CENTER
⚡ GETTING STARTED
Step 1 — Select your firm tab (Deriv, FTMO, The5ers, Other)
Step 2 — Go to Upload Trades and upload your MT5 HTML report or CSV
Step 3 — View stats in Overview, trades in History, insights in Analysis
Step 4 — Journal every trade with emotion tracking for psychology insights
📤 HOW TO EXPORT FROM MT5
MT5 TRADE HISTORY EXPORT
1. Open MT5 → History tab at the bottom
2. Right-click → Custom Period → select date range → OK
3. Right-click again → Report → Save as HTML file
4. Upload the HTML in your firm's Upload Trades tab
⚠ Supported: HTML · CSV · XML · PDF · Screenshots (AI-powered)
